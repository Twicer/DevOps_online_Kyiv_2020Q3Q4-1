<b>Завдання 1</b> <br/>

Для моделювання мережі із чотирьма підмережами було обрано один маршрутизатор із 4 портами для оптоволокна та 4 комутатори, які з'єднані через FastEthernet  із 5 ПК кожен відповідно <br/>
